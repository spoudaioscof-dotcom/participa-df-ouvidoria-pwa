# Roteiro de Demonstração – Participa DF Ouvidoria Multimodal

## Objetivo da demo
Demonstrar o registro de uma manifestação por texto ou áudio,
com geração automática de protocolo, respeitando acessibilidade
e LGPD.

## Passo a passo
1. Abrir a PWA do Participa DF – Ouvidoria
2. Inserir texto ou gravar áudio
3. Selecionar opção de anonimato (opcional)
4. Aceitar o consentimento LGPD
5. Enviar a manifestação
6. Visualizar o número de protocolo gerado automaticamente

## Observações
- A demo foi projetada para funcionar mesmo em ambientes
  de conectividade limitada.
- O áudio é convertido em texto de forma simulada no MVP.
